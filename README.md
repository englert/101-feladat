# 101 Gyakorlat Python Alapokhoz

## Indoklás
A 101 Gyakorlat önellenőrző eszközként szolgál a Python alapjainak tanulásához, beleértve az adattípusokat, operátorokat és felhasználó által definiált függvények írását. Vannak gyakorlatok gyűjtemények gyakorlásához is, mint például listák, szótárak és halmazok.

## A 101 Gyakorlat használata online (nincs beállítás/telepítés, mert hosztolva van)
1. Látogass el a https://colab.research.google.com/github/ryanorsinger/101-exercises/blob/main/101-exercises.ipynb címre
2. Mentsd el a munkádat a Google Drive-ba a Fájl menüben a "Másolat mentése Drive-ba" lehetőség kiválasztásával

## A 101 Gyakorlat használata helyileg (gyorsabban fog futni és hamarabb betölt)
- Töltsd le a repository .zip fájlját, vagy végezd el a `git clone git@github.com:ryanorsinger/101-exercises.git` parancsot
- A `101-exercises.ipynb` az interaktív Jupyter notebook. Ha segítségre van szükséged a Jupyter telepítéséhez, látogass el a https://jupyter-notebook-beginner-guide.readthedocs.io/en/latest/ oldalra
- A `101-exercises.py` egy Python szkript, amely ugyanazokat az utasításokat és ellenőrzéseket tartalmazza (ha esetleg jobban szereted a .py szkripteket)
